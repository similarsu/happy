# lombok

## 使用

```
1、安装插件
2、在Settings设置页面，我们点击Build，Execution，Deployment-->选择Compiler-->选中Annotation Processors，然后在右侧勾选Enable annotation processing即可
3、引入依赖
4、点击View-->Tool Windows-->Structure，便可以看到类中所有的方法了，这些都是lombok帮我自动生成的
```
